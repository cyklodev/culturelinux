# Snap
## Install 
```
# apt install snapd 
# systemctl enable --now snapd.service snapd.socket
# systemctl status snapd.service snapd.socket
# source /etc/profile.d/apps-bin-path.sh
```

## Use
### install
```
# snap install hello-world
```

### list installed
```
# snap list
```
### search
```
# snap find microk8s
```
